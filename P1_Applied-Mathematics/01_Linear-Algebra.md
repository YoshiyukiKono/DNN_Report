#### 1-1-6
##### 1-1-6-1 行列の「逆数」のようなもの

行基本変形 = 左から複数の行列を掛ける。

変形に対応する行列の部分を先に計算したものをつくる（逆行列）。

逆行列を基の式に掛けることで、左辺は打ち消しあい、変数項のみになる。右辺が解として計算される

逆数をかけることで、１になる（スカラーの特定の数を同じ数で割ることで１になる、「割り算」と似ている）。

Ａの逆元：Ａを無効にする（１になる）

単位行列:掛けても相手が変化しない（スカラーにおける「１」）
```
[[1, 0],
 [0, 1]]
```

##### 1-1-6-2 単位行列と逆行列

単位行列: I (前から掛けても、後ろから掛けても、元の行列に影響を与えない)
逆行列: A^-1（Ａインバース）
```
A * A^-1 = A^-1 * A = I
```
##### 1-1-6-3 逆行列の求め方

左辺に単位行列を掛けることで、左右の形式を同じにする。

左右の係数の行列に同じ行基本変形を実行。

ガウスの掃出し法：拡大行列を使って、行基本変形を単位行列に対して行う

#### 1-1-7-2 行列式の特徴

行列式:Determinant = 二つのベクトルで作られる平行四辺形の面積
```
| a b |
| c d |
=
| v1 |
| v2 |
```

同じベクトルを含んでいると0になる
```
| v1|
| ..|
| w |
| ..|
| w |
| ..|
| vn|
= 0
```

以下、「線形性」の特徴（一次=線形 関数のような特徴）

１つのベクトルがλ倍さ
れると行列式はλ倍される
```
| v1 |
| .. |
| λvi|
| .. |
| vn|
=　λ *
| v1 |
| .. |
| vi |
| .. |
| vn |
```

他の成分が全部同じでi番目のベクトルだけが違った場合、
行列式の足し合わせになる
```
| v1 |
| .. |
| vi + w |
| .. |
| vn |
=
| v1 |
| .. |
| vi |
| .. |
| vn |
+
| v1 |
| .. |
| w  |
| .. |
| vn |
```
(上記の３つの法則から下記の法則が導出される)
行を入れ替えると符号が変わる
```
| v1 |
| .. |
| vs |
| .. |
| vt |
| .. |
| vn |
=
  | v1 |
  | .. |
  | vt |
- | .. |
  | vs |
  | .. |
  | vn |
```

３つ以上のベクトルからできている行列式は展開できる
- v1 = (a,b,c)
- v2 = (d,e,f)
- v3 = (g,h,i)
```
| v1 |
| v2 |
| v3 |
=
| a b c |
| 0 e f |
| 0 h i |
=
| 0 b c |
| d e f |
| 0 h i |
=
| 0 b c |
| 0 e f |
| g h i |
=
  a | e f |
    | h i |
- d | b c |
    | h i |
+ g | b c |
    | e f |
```
（２項目：成分を一度入れ替えると符号が変わる。３項目：二回入れ替えると結果同じ符号）
（３成分の行列式＝）体積からの類推：面積と高さに分解
